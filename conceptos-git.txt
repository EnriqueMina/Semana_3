Repositorio:
Es un espacio donde se almacenan los archivos y cambios de un proyecto. Permite organizar el trabajo y llevar un control de las versiones realizadas durante el desarrollo.

Commit:
Es un registro de los cambios realizados en un proyecto. Cada commit guarda una versión específica del trabajo con un mensaje descriptivo.

Push:
Es el comando que permite enviar los cambios del repositorio local hacia GitHub u otro repositorio remoto. Sirve para actualizar el proyecto en línea.
